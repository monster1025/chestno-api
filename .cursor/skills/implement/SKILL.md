# /implement — Реализация по плану

## Процесс

Для каждого шага из `plans/<task-id>-<date>.md`:

1. **Dev** — разработчик реализует шаг
2. **Reviewer** — `ml-agent-reviewer` проверяет код
3. **QA** — `ml-agent-qa` ищет edge cases
4. **Повторять** пока reviewer и QA не подтвердят

## Завершение

- Переключить статус плана на **Готово**
- Вызвать `/git-commit` для фиксации изменений
